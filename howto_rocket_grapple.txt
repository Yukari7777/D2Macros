알려진 요구사항
- 캐릭터가 정지해있어야 함
- 매크로중 마우스를 움직여선 안됨
- 매크로중 피격을 받아 플린칭이 걸려선 안됨
- 인게임 감도와 스크립트의 감도 값 설정이 동일할 것
- 과잉이 3스택일 것
- 수류탄 키가 Q일 것
- 해상도 1920*1080 (1080p)
- 사진과 같은 로켓 (카운터메스, 충격케이스, 상쇄개머리판, 과잉)

맞출 필요 없는 요소
- 로켓 걸작(본인 재장전 걸작임)
- 마우스 DPI(마우스 감도 100, 800에서 동일하게 작동)
- FOV(55나 105나 동일하게 작동)

불확실한 것
- 스크립트의 상수(CONSTANT)값 268(67 * 4)
    - 본인의 테스트 환경인 67(px) * 4(인게임 감도)로 유도된 숫자. 특히, 67에 대해서 미세하게 조정이 필요할 수 있음
    - https://www.youtube.com/watch?v=ItN-K-WSCkM 참고
- 반동방향 100?
- 안정성 92 이상? 더 낮을 경우 실패확률이 증가하는 느낌인듯. 과잉 강화퍽 넣으면 성공확률이 더 증가하는지 여부 확인 필요
- 투사체 속도 51 이하?


추가 참고 사항
- 수류탄 사용시 장전조각 끼면 좋음.
- 에임을 최대로 올려서 수직으로 올라가고자 하는 것이라면, 로켓 종류는 상관 없다. 이경우, 아무 로켓 들고 클릭 + 수류탄만 누르면 됨.
- 로켓 4퍽을 강화연쇄로 하면 최대 탄창이 1개 늘어남. 
    - 강화냉탄을 박으면 안정성이 더 오르므로 성공확률이 더 증가할 수도 있음
